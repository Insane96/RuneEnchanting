# Rune Enchanting
A NeoForge mod for Minecraft 1.21.1 that replaces the vanilla enchanting system with a rune-based, exploration-driven alternative.

## Overview
Instead of using an enchanting table and experience levels, items are enhanced by socketing Runes found through exploration and gameplay. Runes can modify mining speed, attack damage, durability, protection, projectile behavior, and apply per-tick effects — covering the same breadth as vanilla enchantments but without the RNG-at-a-table experience.

The mod disables vanilla XP gain, the experience bar, and the enchanting table entirely, fully replacing the system.
